# MICRO VIDEOS ADMIN

Status: em desenvolvimento

## Pré-requisitos
- Nodejs
- Docker

### Tecnologias
- Typescript

#### 1. Instale as dependências

```sh
$ npm install # ou yarn
```

#### 2. Execute o projeto localmente

```sh
$ docker compose up
```

#### (opcional) Acesse o container

```sh
$ docker exec -it micro-videos-admin bash
```

#### 3. Executar testes

```sh
$ npm run test
```
